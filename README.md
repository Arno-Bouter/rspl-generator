# RSPL Generator Pro

Maritime Spare Parts List Generator with Claude AI analysis.

## Features

- ✅ PDF parts manual upload & AI analysis
- ✅ Online manual search (fallback)
- ✅ Real article numbers extraction
- ✅ English maritime terminology
- ✅ Proper XLSX export
- ✅ Logo with zoom functionality
- ✅ 0-6 year vessel operation recommendations

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Ready for Netlify deployment via GitHub!
