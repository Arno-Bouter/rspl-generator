# RSPL Generator Pro - Vite Edition

Geavanceerde Recommended Spare Parts List Generator voor Grootkeuken Apparatuur.

## Versie: FINALE MET XLSX EXPORT

Alles wat je nodig hebt om direct te deployen!

## Features

✅ PDF Upload & Parsing  
✅ Online Parts Manual Search  
✅ AI Onderdeel Selectie  
✅ **XLSX EXPORT** (echte Excel files!)  
✅ Alle 24 Kolommen (automatisch ingevuld)  
✅ Job History  
✅ Real-time Progress Tracking  
✅ Professional Dark UI  

## Ondersteunde Apparatuur

- Ovens (Rational, Electrolux, etc.)
- Fryers
- Griddles
- Steamers
- Mixers
- Dishwashers

## Snelle Start

1. `npm install` - Install dependencies
2. `npm run dev` - Run locally
3. Open http://localhost:5173

## Build & Deploy

```bash
npm run build
```

Deploy op Netlify via GitHub!

## File Structuur

- `src/App.jsx` - Main React component (MET XLSX EXPORT!)
- `src/main.jsx` - React entry point
- `src/index.css` - Styling
- `vite.config.js` - Vite configuration
- `netlify.toml` - Netlify deployment config

## Excel Export

Click "Download RSPL als Excel (XLSX)" to export all data with:
- Spare part names
- Supplier part numbers
- CAGE codes
- HS codes
- COO
- Quantities
- All 22 columns complete and ready for maritime use!
